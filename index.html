Informed preference of patients for
over-the-counter medication
By Yusaf Khaliq
About
This website was created to display in infographic form; the results from a series of exit interviews carried out at three pharmacies. The aims of the interviews were to identify where people get information from for OTC medicines. Moreover, discover the impact pharmacy intervention can have by looking at intentions prior to visiting the pharmacy and comparing these to purchase outcomes.
How to use the website
This website works on modern browsers, and heavily relies on JavaScript which will need to be enabled if not already. JavaScript is disabled.
1. At the very bottom of the page is the infographic display.
2. Below are sections which allow you to filter and highlight the infographic displayed at the bottom.
3. The section entitled "Results / preset filters" is simple to use and gives an overview of the results. Within the text are
buttons
which are programmed with preset filters, and when clicked automatically changes the infographic.
4. The other sections are more complex and should only be used for custom filtering. Though not necessary, the "Choose columns/rows" section allows you to divide the infographic into sections based on pharmacy and demographics. The "Highlight" section allows you to highlight the infographic at the bottom based on responses to the question. To further filter the infographic, at the bottom is an
Add Filter
button. Clicking this will create a new section allowing you to filter responses based on the questions asked. The
Collapse All
button closes all sections taking up less room. The
Reset Filters
button allows you to reset all the filters added. The
Reset All
button allows you to reset the columns/rows and filters added.
